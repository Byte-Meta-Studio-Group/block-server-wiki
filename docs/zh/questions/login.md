# 登录相关问题

## 身份验证服务器已关闭或宕机，请联系管理员查看. 或稍等再试

您是否使用了正版登录或者外置登录？

如果没有的话，请通过[此处](../#注册皮肤站)重新设置

否则，请尝试打开[皮肤站网页](https://skin.bytemetasg.top/)

如果以上手段仍无效，请[联系管理员](../contact/)解决

如果有的话，请更换回来，否则，请[联系管理员](../contact/)核对后解决
